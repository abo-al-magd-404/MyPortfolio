# Portfolio Project

## Overview
This is my personal portfolio website, built to showcase my projects, skills, and experiences as a **Full-Stack Developer (MERN Stack)**. The website is fully responsive, modern, and interactive, giving a clear overview of my work and achievements.

## Features
- **Responsive Design:** Works perfectly on mobile, tablet, and desktop screens.  
- **Skills Showcase:** Highlights technical skills, tools, and technologies I work with.  
- **Projects Section:** Interactive cards displaying my key projects with live demos and GitHub links.  
- **Contact Section:** A fully functional contact form to reach me directly.  
- **Animations:** Smooth transitions and animations for better user experience (using Framer Motion).  
- **Dark Mode Ready:** Modern styling with Tailwind CSS.  

## Tech Stack
- **Frontend:** React.js, Next.js, Tailwind CSS, TypeScript, Framer Motion  
- **Backend (if any):** Node.js, Express.js  
- **Database (if used):** MongoDB  
- **State Management:** Redux Toolkit  
- **Other Tools:** Git, Vercel for deployment
